## Fluid Contact

# Requirements
- Node.js
- npm

1. Open your Command Prompt
2. Clone the Project: git clone https://github.com/makozi/Fluid-Contact.git
3. Navigate to the location of the folder: cd Fluid-Contact
4. npm install
5. npm run build
6. npm server.js
7. Open your browser at http://localhost:5000
